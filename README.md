# 仓库管理系统简介 python790

## 项目概述

仓库管理系统是一款基于Python语言的Web应用程序，采用Django框架进行开发，前端使用了Vue技术，数据库方面选用了MySQL。系统遵循B/S架构，旨在帮助企业和组织高效管理仓库的出入库流程，优化库存管理。

## 技术栈

- **后端**: Python, Django
- **前端**: Vue
- **数据库**: MySQL
- **架构**: B/S

## 功能模块

### 职工功能

- **首页**: 查看系统概况
- **个人中心**: 个性化设置与信息管理
- **库房信息管理**: 管理库房基本信息
- **商品信息管理**: 管理商品详细信息
- **出库信息管理**: 记录和管理商品出库信息
- **入库信息管理**: 记录和管理商品入库信息

### 管理员功能

- **首页**: 查看系统总览
- **个人中心**: 管理个人信息
- **职工管理**: 管理职工账号和权限
- **库房信息管理**: 管理所有库房信息
- **商品信息管理**: 统一管理商品资料
- **入库信息管理**: 监控和管理所有入库信息
- **出库信息管理**: 监控和管理所有出库信息

## 系统角色

- **职工**: 负责日常出入库操作及信息管理
- **管理员**: 负责系统维护、权限分配及综合信息管理

## 运行环境

- **操作系统**: 兼容主流操作系统
- **Web服务器**: 建议 Nginx 或 Apache
- **Python版本**: 建议 Python 3.x
- **数据库**: MySQL 5.x 或更高版本
